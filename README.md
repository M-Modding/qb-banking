# qb-banking
A redesign of QB-Banking. More scripts in our Discord: https://discord.gg/5sfVn6XmAJ


Features
- Handles all player interaction with bank/job/gang/shared accounts
- ATM and bank card integration
- Shared accounts between players
- Auto creation of job/gang accounts on bank first open
- Boss-only access to job/gang accounts
- Documentation: https://docs.qbcore.org/qbcore-documentation/qbcore-resources/qb-banking

# qb-banking
QB-Bankingin uudelleensuunnittelu. Lisää skriptejä Discordissamme: https://discord.gg/5sfVn6XmAJ

Ominaisuudet
- Käsittelee kaiken pelaajan vuorovaikutuksen pankki-/työ-/jengi-/jaettujen tilien kanssa
- Pankkiautomaatin ja pankkikortin integrointi
- Jaetut tilit pelaajien kesken
- Työ-/jengitilien automaattinen luonti pankin ensimmäisen avauksen yhteydessä
- Vain pomolla pääsy työ-/jengitileille
- Dokumentaatio: https://docs.qbcore.org/qbcore-documentation/qbcore-resources/qb-banking
